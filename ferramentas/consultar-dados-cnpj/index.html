Sobre os dados consultados

Os dados exibidos nesta consulta de CNPJ, quando disponíveis, são fornecidos gratuitamente pelo projeto OpenCNPJ, permitindo consulta de dados reais de CNPJs.

As informações são atualizadas uma vez por mês e refletem dados públicos da Receita Federal. Por isso, pode haver leve defasagem entre uma atualização e outra. Para conferência oficial, verifique diretamente no site do Gov.br.

Se o projeto te ajudou, considere dar aquela força! As opções de apoio estão disponíveis na página oficial.
