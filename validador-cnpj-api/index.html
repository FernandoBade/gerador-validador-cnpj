Sobre os dados consultados

Os dados retornados vêm da Receita Federal. A consulta exige aceitar o reCAPTCHA para cada pedido. Por questões legais, não armazenamos os dados retornados e apenas exibimos a resposta diretamente no navegador do usuário.

Perguntas sobre a API de validação

Como fazer a validação de CNPJ via API?

Utilize o endpoint documentado para enviar o número ou lista de CNPJs. A resposta inclui status de validação e indica quando seguir para a consulta de CNPJ grátis na Receita Federal.

Por que consultar CNPJ na Receita Federal mesmo após a checagem?

A checagem de CNPJ garante integridade matemática, mas apenas a Receita Federal confirma situação cadastral e eventos. Combine as etapas para auditorias completas.
